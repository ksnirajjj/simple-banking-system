# Project Name: Simple Banking App

# Overview:
This is a simple banking system that I built using C++. It uses json file to store data. It allows you to:
1. Create account
2. Login
3. Deposit
4. Withdraw
5. Transfer
6. Freeze accounts through admin login
7. Make changes to account through admin login

# Requirements:
1. C++17 or later
2. nlohmann/json library

# How to comiple:
g++ -std=c++17 functions.cpp main.cpp -o main

# How to run:
./main
